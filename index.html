Tag 'Form' e seus atributos

Tag 'Form"

A tag 'form' Cria uma área onde certos inputs serão enviados juntos, ela também necessária para o funcionamento da tag button.

Method

O método 'get' envia as informações pelo URL, o que o torna não-seguro para o envio de senhar.

Já o método 'post' envia uma requisição, tornando-o apropriado para senhas.

Target

O target '_blank' envia as informações para uma nova página.

Enquanto o método '_self' modifica a própia página.

A ideia do target é modificar se a informação trafega em outra página ou não.

Autocomplete

O autocomplete 'off' desabilita a autocompletação que é armazenda pelo navegador.

Já o autocomplete 'on' guarda a informação digitada no campo quando sair e voltar á pagina.

Prefixo 'ON'

O prefixo 'on' funciona como um comando de eventos ligado ao JavaScript.

  • Onsubmit

    • O evento 'onsubmit' funciona quando algo é enviado, ou seja, nesse caso quando o botão é clicado, ele ativa o evento 'alert' (JS). Esse recurso pode ser usado para validação de formulários entre outros.

  • Onclick

    • O evento 'onclick' funciona identificando quando algo é clicado.

Tag 'Input' e seus tipos e atributos

Tag 'input'

A tag 'input' abre uma caixa para a inserção de informações como letras e números, podendo ter diversos tipos e atributos.

Text

O input 'text' permite a inserção de letras e números dentro da caixa de texto.

Number

O input 'number permite somente a inserção de números dentro da caixa de texto, adicionando também, por padrão, setas ao lado da caixa para a escolha dos números.

  • Atributo 'min'

    • Observe que na caixa de texto acima o número pode ser negativo, isso não é muito bom quando a contagem deve ser de números naturais, como idades, para isso, o atributo 'min' funciona definindo um limite mínimo para a contagem, como 0 por exemplo.

  • Atributo 'max'

    • Da mesma forma o atributo 'max' define um limite máximo para a contagem, como 10 por exemplo.

    Podendo os dois atributos serem combinados também.

  • Atributo 'step'

    • Naturalmente o contador progride acontagem de 1 em 1, isso pode ser modificado através do atributo 'step' o qual define uma progressão para o campo numérico, por exemplo 5.

    Podendo os dois atributos serem combinados também.

Password

O input 'password' permite a mesma inserção de textos que o 'text' porém eles são escondidos por caracteres para que seja seguro a inserção de senhas.

Button

O input 'button' cria um botão virtual que pode ser clicado para diversas funções.

Diversos atributos podem ser aplicados como o 'value' para dar um nome ao botão.

Existe também uma tag própia chamada button que poderá ser abordada mais tarde. Semânticamente falando, é melhor usar a tag própia do que o 'input button', abaixo está a tag button para comparação.

Range

O input 'Range' cria um campo scroll, muito útil em diversos casos.

  • Atributos 'min', 'max' e 'value'

    • Os atributos 'min' e 'max' também podem ser utilizados aqui. Embora não sejam aparentes, o mínimo e o máximo estão definidos como 0 e 100

    Vale ressaltar o uso do atributo 'value', este define a posição inicial da barra, definindo um valor inicial a ela. Nesse caso o value é de 20.

  • Atributo 'Step'

    • Esse atributo também pode ser utilizado aqui, seu uso é interessante pois parece criar "marcos" ao longo da barra facilitando seu uso em certas ocasiões, já que naturalmente a contagem é feita de 1 em 1, deixando o movimento bastante suave.

    Aqui o step está definido como 25, e o máximo como 100, dessa forma, há 4 posições possíveis para que a bolinha seja colocada ao longo da barra.

Color

O input 'color' permite a criação de um campo para a escolha de cores, o qual está a mercê do tipo de janela exibida por cada navegador.

Email

O input 'email' exibe uma caixa de texto com a finalidade de inserção de emails, com isso essa caixa exige que o caractere @ esteja presente.

Esse é um campo muito padrão do navegador, e geralmente é substiuído por outros tipos, por questões de problemas de compatibilidade e etc.

URL

O input 'url' exibe uma caixa de texto com a finalidade de inserção de url, dessa forma, a caixa exige que esteja presente https:// dentro do campo.

Date

O input 'date' permite a criação de um campo para a escolha de datas, o qual está a mercê da linguagem escolhida no navegados para a exibição da ordem dos dia, mês e ano, e do navegador escolhido para o tipo de janela que será exibida.

Week

O input 'week' é bem diferente, pois permite a escolha de uma semana do ano, o qual novamente está a mercê do navegador escolhido para o tipo de janela que será exibida.

ATENÇÃO esse input é tão pouco utilizado que ele sequer funciona no firefox

Month

O input 'month' é o mesmo caso do anterior e permite a escolha de um mês do ano.

ATENÇÃO esse input também sequer funciona no firefox

Hidden

O input 'hidden' cria uma caixa escondida para o usuário que envia informações "por debaixo dos panos".

File

O input 'file' cria um campo para enviar arquivos.

  • Atributo 'multiple'

    • O atributo 'multiple' permite que a escolha de arquivos seja multipla, simples assim.

Search

O input 'search' cria uma caixa de busca que permite a inserção de textos normalmente, seu diferencial é que ele permite que o texto seja todo apagado através de um botão, muito útil em diversos casos.

ATENÇÃO esse input infelizmente só funciona no chrome, sendo necessário a criação de um campo personalizado para outros navegadores

Checkbox

O input 'checkbox' cria uma caixa de seleção de múltipla escolha, ou seja, várias podem ser escolhidas ao mesmo tempo.

  • Atributo 'name'

    • O atributo 'name' permite que os itens sejam "agrupados", dessa forma, permitindo que a funcionalidade de múltipla escolha funcione dentro de uma categoria e em diferentes categorias ao mesmo tempo.



    ATENÇÃO para que esse atributo seja entendido como uma lista pelo servidor back-end é necessário que colchetes "[]" sejam adicionados ao final do nome e o formulário enviado pelo método 'post'.

  • Atributo 'value'

    • O atributo 'value' permite que os itens sejam "identificados", dessa forma, selecionando qual informação será enviada para o servidor. Nesse caso, cada caixa selecionada irá enviar um valor igual ao seu nome.




  • Atributo 'disabled'

    • O atributo 'disabled' permite que as caixas de seleção sejam desabilitas.

Radio

O input 'radio' cria uma caixa de seleção de única escolha desde que tenham o mesmo nome, ou seja, desde que pertençam ao mesmo grupo.

  • Atributo 'name'

    • O atributo 'name' permite que os itens sejam "agrupados", dessa forma, permitindo que a funcionalidade de única escolha funcione dentro de uma categoria e em diferentes categorias ao mesmo tempo.



  • Atributo 'value'

    • O atributo 'value' permite que os itens sejam "identificados", dessa forma, selecionando qual informação será enviada para o servidor. Nesse caso, cada caixa selecionada irá enviar um valor igual ao seu nome. Ou seja, ou será 1, ou será 2 ou então sera 3.




  • Atributo 'disabled'

    • O atributo 'disabled' permite que as caixas de seleção sejam desabilitas.

Tag 'br'

Tag 'br'

A tag 'br' faz uma quebra de linha e é geralmente utilizada no fim de uma linha. A exemplo < br >
a tag foi usada aqui para fazer a quebra, no código esse texto é contínuo, e na web ele aparece como uma quebra de linha.

Tag 'Button' e seus Tipos

Tag 'button'

A tag 'button' cria uma caixa clicável para diversas finalidades. Ele possui 4 tipos diferentes acessáveis através do atributo 'type' e é altamente customizável através do JS.

Button

O atributo 'button' simplesmente cria um botão e nada mais, ele somente ganhará uma função através do JavaScript.



Reset

O atributo 'reset' limpa os campos do formulário no qual ele está dentro.

Submit

O atributo 'submit' envia as informações dos campos do formulário no qual ele está dentro. É um atributo ligado ao JS através do evento 'OnSubmit'

Esse atributo também identifica quando a tecla enter do teclado é pressionada, e age como se o botão fosse clicado com o mouse.

ATENÇÃO É muito importante fazer a validação no back-end também não somente no front-end, porque muitas vezes o front-end pode ser manipulado através do console do navegador.

Tag 'Select' e seus Tipos

Tag 'select'

A tag 'select' cria uma caixa de seleção com diferentes opções para escolha

Tag filha 'option'

A tag filha 'option' permite a criação da lista com diferentes itens dentro da tag 'select'.

  • Atributo Multiple

    • Esse atributo é aplicado à Tag 'select' e permite que várias opções sejam selecionadas usando a tecla "ctrl" ou "Shift" do teclado
